# CyberLaw Guardian

CyberLaw Guardian adalah aplikasi Flutter untuk pembelajaran keamanan siber, hukum cyber, dan kompetisi CTF.
Aplikasi ini dibuat untuk membantu pengguna belajar tentang ancaman keamanan, etika hacking, dan hukum cyber melalui fitur interaktif dan kumpulan alat praktis.

## Fitur Utama

- **Autentikasi Pengguna**: login dan registrasi untuk menjaga akses personal.
- **Beranda Interaktif**: ringkasan fitur utama dan navigasi ke modul edukasi.
- **Cyber Law**: artikel hukum cyber lengkap dan detail untuk memahami regulasi dan etika digital.
- **Security Awareness**: wawasan keamanan yang membantu pengguna mengenali ancaman siber nyata.
- **Security Tools**: koleksi alat keamanan seperti:
  - Base64 encoder/decoder
  - Binary decoder
  - Hash generator
  - Password checker
  - Phishing checker
  - Data leak scanner
- **CTF Challenges**: tantangan belajar keamanan dalam kategori:
  - Crypto challenges
  - Forensic challenges
  - Reverse engineering
  - Web challenge
  - Cyber lab
- **Community**: ruang untuk melihat bagian komunitas dan berbagi informasi.
- **Leaderboard**: peringkat pengguna untuk memotivasi partisipasi dan pencapaian.
- **Report**: fitur laporan untuk menganalisis dan mendokumentasikan hasil.

## Platform

Dikembangkan dengan Flutter, proyek ini mendukung target mobile dan dapat dijalankan pada Android, iOS, web, serta desktop bila dikonfigurasi.

## Cara Menjalankan

1. Pastikan Flutter sudah terpasang pada sistem Anda.
2. Buka folder proyek di terminal.
3. Jalankan:
```bash
git clone https://github.com/AgA-w4hYu/CyberLaw-Project.git
cd cyberlaw
flutter pub get
flutter run
```
